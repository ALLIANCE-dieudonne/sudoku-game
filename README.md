Welcome to a fun and engaging Sudoku experience!

## Description

This project presents a visually appealing Sudoku game built using Java and JavaFX. Challenge yourself with puzzles of varying levels, save and load progress, and enjoy a user-friendly interface.

## Features

Intuitive interface designed for easy gameplay
Multiple difficulty levels to cater to different skill sets
Option to highlight possible numbers for a better solving experience
Save and load game functionality to resume progress later
Clear visual cues for correct and incorrect entries
Option to check solution for guidance
## Technologies Used

Java: Core programming language
JavaFX: Powerful framework for creating graphical user interfaces
## Getting Started

Prerequisites:
Ensure you have Java and JavaFX installed on your system.
Download:
Clone or download the repository to your local machine.
Run:
Compile and run the Java program.
## Instructions

Select a Difficulty Level: Choose a puzzle that suits your Sudoku expertise.
Fill the Grid: Enter numbers 1-9 in empty cells, ensuring each row, column, and 3x3 subgrid contains unique digits.
Hints: Use optional highlighting for assistance.
Check Solution: Verify your answers if needed.
Save and Load: Preserve your progress for a later session.
